# 1.0.1

Initial public release

- Added AppIcon widget for rendering adaptive icons (Material / Cupertino).
- Added AppIconController to explicitly control icon animations.
- Introduced a curated set of reusable icon animations:
  - pulse
  - pop
  - heartBeat
  - tada
  - wobble
  - swing
  - zoomIn / zoomOut
  - fadeIn / fadeOut
  - shakeX / shakeY
  - rotateLeft / rotateRight
  - flipXLeft / flipXRight
  - flipYUp / flipYDown
  - particle
- Added support for one-shot and repeatable animations.
- Added animated icon transitions via playWithIconTransition.
- Ensured all animations return to their initial visual state.
- Provided usage examples and documentation.
